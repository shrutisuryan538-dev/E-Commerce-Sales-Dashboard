# E-Commerce-Sales-Dashboard

## Project Overview
This project analyzes ecommerce sales data to understand overall business performance, customer purchasing behavior, and profit trends. The dashboard helps identify key sales drivers, profitable product categories, and regional performance to support data-driven business decisions.

## Tools & Technologies Used
Microsoft Power BI
Excel Dataset
Data Cleaning
Data Visualization

## Dataset
The dataset contains ecommerce transaction details such as:
Order Amount
Quantity Sold
Profit
Customer Name
State
Product Category & Sub-Category
Payment Mode
Order Date

## Analysis Performed
Total sales, quantity, profit, and average order value analysis
State-wise sales performance analysis
Monthly profit trend analysis
Payment mode distribution analysis
Product category and sub-category profitability analysis
Customer-wise sales contribution analysis

## Key Insights
Maharashtra and Madhya Pradesh contribute significantly to total sales.
Clothing category accounts for the largest share of quantity sold.
Printers and Bookcases generate the highest profits among sub-categories.
Cash on Delivery (COD) is the most commonly used payment method.
Profit varies significantly across months, indicating seasonal sales patterns.

## Dashboard Features
Interactive state filter
Quarter-based filtering
KPI cards for quick business overview
Visual analysis for sales, profit, and product performance

## Purpose of the Project
The purpose of this project is to transform raw ecommerce data into meaningful insights that help businesses improve sales strategies, product planning, and customer targeting.
